About me

Built on
clean code.

I'm a Java Full Stack Developer based in Hyderabad, with a strong focus on backend engineering. My core stack is Java, Spring Boot, Spring Security, Hibernate, and JPA — but more than the tools, I care about the decisions underneath them: how you model entities and relationships, where you enforce constraints, and how those choices hold up once real traffic and real edge cases show up.

During my time at Amiran Soft Tech, I worked across full-stack modules in an Agile team — tracing production defects back to root cause through code review and testing, and contributing to Git workflow improvements that made sprints noticeably smoother. It gave me a clear picture of how individual code decisions ripple into team velocity.

Outside of work, I build complete systems end-to-end, not isolated features. A multi-role hospital management platform with 21+ REST APIs and RBAC across 10 user types. A banking system designed around strict ACID guarantees. A live business website running on Firebase, handling real users. What draws me to backend work is the depth that only appears once a system is actually running: the edge cases, the failure modes, the things that don't show up in a tutorial.
